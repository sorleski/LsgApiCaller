Setup - How To

  1. For SS, to run locally against your local websphere, be sure to change the XCRSF filter to run locally as shown below:
  2. To add a new form/controller rest caller do the following
    • Open /js/[appName]Controllers.js, example SS = ssControllers.js
    • Copy one of the existing controller methods and change to meet the needs of the endpoint you are trying to test
    • Create a corresponding html template page in templates/[APP_NAME]/[controller_endpoint_name].html
    • Open /js/app.js, copy one of the existing "when" clauses and change to match the endpoint and html page you just created.
    • Open index.html and change or add your endpoint under the appropriate menu system.
    • Please keep with the naming convention that is currently in use throughout the app. It is easy to catch on to and shouldn't be difficult to figure out.
